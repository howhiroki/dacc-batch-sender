# 🚀 DACC Batch Sender (Testnet Tool)

Live tool:  
👉 https://howhiroki.github.io/dacc-batch-sender

---

## 🧠 What This Is

A simple browser-based tool that automates sending small amounts of DACC testnet tokens to multiple wallet addresses.

Built to simplify repetitive transaction tasks on the DAC Inception Testnet.

---

## ⚡ Key Features

- Runs fully in the browser (no downloads or installs)
- Sends DACC tokens to multiple randomly generated addresses
- Adjustable amount per transaction
- Adjustable number of recipients
- Real-time progress tracking
- Transaction hash logging
- Fully client-side (no backend)

---

## 🛠 How It Works

1. Open the live tool link  
2. Paste your **testnet private key**  
3. Set:
   - Amount per wallet (e.g. 0.001 DACC)
   - Number of wallets (e.g. 50–60)
4. Click **Run Batch Send**
5. The tool:
   - Generates wallet addresses
   - Sends transactions sequentially
   - Displays progress + hashes in real time

---

## 🔐 Security

- Runs entirely in your browser
- Private key is never stored or transmitted anywhere
- No backend, no logging, no data collection
- Use **testnet wallets only**
